# AI Agents Instructions

## 目的
エージェント共通の作業方針を定義

## 基本方針
- **会話言語**: 日本語
- **コメント言語**: 日本語
- **エラーメッセージ**: 英語
- **コミット規約**: Conventional Commits

## 判断記録のルール
- 判断内容、代替案、採用理由、前提条件、不確実性の明示

## 開発手順（概要）
1. プロジェクト理解 (`README.md` 参照)
2. 変更実装 (PowerShell)
3. 動作確認 (Windows 環境が必要)

## セキュリティ / 機密情報
- 認証情報のコミット禁止
- ログへの機密情報出力禁止

## リポジトリ固有
- Windows 環境での PowerShell 実行を前提とする。
- `$steam_username` はユーザーごとに設定が必要。
